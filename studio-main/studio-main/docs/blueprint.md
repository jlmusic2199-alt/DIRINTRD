# **App Name**: DIPRINT RD Tracker

## Core Features:

- Task Table: Implement a Trello-like task table to manage print jobs internally.
- Job Details: Detailed view for each job with information such as client, specifications, deadline, and status.
- Status Updates: Allow internal users to update the status of each job (e.g., "Received", "In Production", "Quality Control", "Shipped").
- Client Link Generation: Generate a unique tracking link for each job to share with the client.
- Client View: Clients can view the status of their job using the link, without needing an account.
- Google Email Connection: Connect to Google Email like ClickUp to manage communications and updates.

## Style Guidelines:

- Primary color: A vibrant cyan (#42C2F5), evoking a sense of precision and modernity associated with printing technology.
- Background color: A very light cyan (#E5F6FD), for a clean and professional look.
- Accent color: A strong, analogous blue (#4281F5), used to highlight important interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use simple, clear icons to represent different job statuses.
- Clean and organized layout for the task table, prioritizing readability and ease of use.
- Subtle animations to indicate changes in job status.
